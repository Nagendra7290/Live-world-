## 🧠 Project Overview  

The **World Population Live Dashboard** is a real-time Business Intelligence solution designed to transform raw global population data into decision-ready insights.

This project demonstrates **end-to-end data workflow**, including:

- 🌐 Live data extraction  
- 🔄 Data transformation & modeling  
- 📊 Advanced dashboard design  
- 📈 KPI engineering  
- 🎯 Insight generation  

---

## ⚙️ Data Engineering (Live Data Pipeline)  

One of the key highlights of this project is **automated real-time data extraction**.

### 🔗 Data Source  
https://www.worldometers.info/world-population/

### 🛠 Data Extraction Method  

Data is dynamically fetched into **Google Sheets** using:

- `IMPORTHTML()`  
- `IMPORTXML()`  
- Data cleaning formulas  
- Structured tabular transformation

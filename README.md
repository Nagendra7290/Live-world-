## Project Overview

The World Population Live Dashboard is a real-time Business Intelligence solution developed to convert continuously updating global population data into structured, decision-ready insights.

This project demonstrates a complete BI lifecycle, including data extraction, transformation, modeling, and visualization in an automated environment.

---

## Key Objectives

- Build a real-time population monitoring system
- Design an automated data pipeline
- Develop KPI-driven analytics framework
- Deliver interactive dashboard experience
- Enable faster decision-making through insights

---

## Data Engineering Architecture

### Data Source
https://www.worldometers.info/world-population/

### Data Extraction Layer

- IMPORTHTML()
- IMPORTXML()
- Web-based structured extraction

### Data Transformation Layer

- Data cleaning and normalization
- Handling missing and inconsistent values
- Converting raw data into structured format
- Creating calculated fields

### Data Pipeline Flow

Web Source → Google Sheets → Looker Studio
